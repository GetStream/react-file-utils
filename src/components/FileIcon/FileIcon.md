### File Icons V1

Small icons (can have custom width & height)

```js
<FileIcon mimeType="application/msword" filename="document.doc" /> <br/>
<FileIcon mimeType="application/vnd.ms-excel" filename="accounting.xls" /> <br/>
<FileIcon mimeType="application/vnd.ms-powerpoint" filename="presentation.ppt" /> <br/>
<FileIcon mimeType="application/pdf" filename="document.pdf" /> <br/>
<FileIcon mimeType="image/png" filename="image.png" /> <br/>
<FileIcon mimeType="text/csv" filename="output.csv" /> <br/>
<FileIcon mimeType="application/x-tar" filename="archive.tar.gz" /> <br/>
<FileIcon mimeType="application/zip" filename="archive.zip" /> <br/>
<FileIcon mimeType="video/mp4" filename="video.mp4" /> <br/>
<FileIcon mimeType="audio/mp3" filename="music.mp3" /> <br/>
<FileIcon mimeType="application/json" filename="config.json" /> <br/>
```

Big icons (can have custom width & height)

```js
<FileIcon mimeType="image/png" filename="image.png" big/> <br/>
```

### File Icons V2

Standard icons (can be big / small / custom width & height)

```js
<FileIcon mimeType="image/png" filename="image.png" version="2" big/> <br/>
<FileIcon mimeType="application/msword" filename="document.doc" version="2" big/> <br/>
<FileIcon mimeType="application/vnd.ms-excel" filename="accounting.xls" version="2" big/> <br/>
<FileIcon mimeType="application/vnd.ms-powerpoint" filename="presentation.ppt" version="2" big/> <br/>
<FileIcon mimeType="application/pdf" filename="document.pdf" version="2" big/> <br/>
<FileIcon mimeType="application/x-tar" filename="archive.tar.gz" version="2" big/> <br/>
<FileIcon mimeType="video/mp4" filename="video.mp4" version="2" big/> <br/>
<FileIcon mimeType="audio/mp3" filename="music.mp3" version="2" big/> <br/>
<FileIcon mimeType="application/json" filename="config.json" version="2" big/> <br/>
<FileIcon mimeType="text/" version="2" big/> <br/>
```

